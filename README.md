# Tracevia
Tracevia is ideal for locating lost devices or keeping track of personal devices with full user consent.

## Getting Started

### Backend Setup
1. Open a terminal and navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the backend server:
   ```bash
   npm run dev
   ```

### Frontend Setup
1. Open a terminal and navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend React app:
   ```bash
   npm start
   ```

### Notes
- Make sure MongoDB is running for backend API.
- Vercel will use the `frontend` folder for deployment.
- All backend API code is in the `backend` folder.
